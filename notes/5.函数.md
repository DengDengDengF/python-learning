### 2.参数传递

```python
//和js规定的一样，对于值类型和引用类型
def test(num, obj):
    num = 2
    obj['key'] = 2
    print('test函数块内部修改后', num, obj)
    return


num = 1
obj = {'key': 1}
test(num, obj)
print('test函数块外部修改后', num, obj)


//不定长参数
def printList(str, *list):
    print(str)
    for item in list:
        print(item)
        print()
    return


str = '123'
printList(str)
printList('str', 1, 2, 3)
```

### 4.匿名函数

```python
用lambda创建匿名函数
sum = lambda arg1, arg2: arg1 + arg2
print('相加后的值', sum(1, 2))
print('相加后的值', sum(10, 20))
```

### 5.变量作用域

```python
全局作用域：函数外，整个程序内访问
局部作用域：函数内，被声明
def AddMoney():
    # python会首先猜测他是个局部变量，如果不声明他是个全局变量，它会报错
    global Money
    Money = Money + 1
    return


print(Money)
AddMoney()
print(Money)
```
